## Instructions

1. Download 10 high-resolution images from Unsplash that match the categories above and rename them as listed.
2. Copy the 2 logo files (logo-black.png, logo-white.png) and favicon.ico from the project description or provided resources.
3. Place all images in the `images` folder.
4. Commit and push your changes to GitHub.
