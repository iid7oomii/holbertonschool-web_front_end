## Image Guidelines

- **Resolution**: High resolution (at least 1920x1080 for landscape, 1080x1080 for portraits)
- **Format**: JPG for photos, PNG for logos
- **Style**: Professional, modern, consistent color palette
- **Licensing**: Use free images from Unsplash (no attribution required for Unsplash images)
